🪷 QR Code — BGC: THE DUAL-BIO SYNERGY

Tạo QR Code cho slide cuối cùng của bài thuyết trình

📋 Hướng dẫn Deploy trang Thank You lên Netlify (Miễn phí)

  1. Truy cập https://app.netlify.com và đăng nhập
  2. Kéo thả thư mục QR_Code (chứa index.html, style.css, script.js) vào vùng deploy
  3. Netlify sẽ tự động deploy và cung cấp URL (ví dụ: https://random-name.netlify.app)
  4. Đổi tên site trong Settings → Site details → Change site name thành tên bạn muốn
  5. Copy URL, dán vào ô bên trên, và bấm "Tạo QR Code"
  6. Tải hình QR Code về, chèn vào slide cuối!

⚠️ Lưu ý: File generate_qr.html này chỉ dùng để tạo QR, KHÔNG cần deploy file này. Chỉ deploy 3 file: index.html, style.css, script.js.
